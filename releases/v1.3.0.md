# v1.3.0 - 手机日历同步

## 新增

- 📱 **手机日历同步**：可选写入 iCloud / Google / Outlook / 网易 163 / 移动 139 等云账户，事件自动同步到手机
- **日历账户选择 Picker**：主界面选择写入哪个账户（按 source 分组，本地日历排最后）
- **首次启动引导**：未配置云账户时延迟 3s 自动弹 3 步配置（163 / 139 可切换）
- **同步帮助**：标题栏右上角「?」按钮随时重看配置步骤
- **失效回退**：选中日历账户被删 / 注销时自动回退到系统默认并提示
- **重复检测优化**：改为只在目标日历内查重（避免跨账户误报）
- **本地日历警告**：选中「本地」时黄色提示「不会同步到手机」

## 改进

- README 优化，加入完整的手机同步章节和 CalDAV 源对比表
- AGENTS.md 末尾新增「手机日历同步」章节

## 下载

下载 `一键日历.app.zip`，解压后把 `一键日历.app` 拖到「应用程序」文件夹即可。

## 配置手机同步

启动 App → 标题栏点「?」→ 按引导 3 步操作（开启 CalDAV / Mac 加账户 / 手机加账户）。推荐用网易 163 个人邮箱，免费稳定。

## 系统要求

macOS 14.0+，首次使用会请求日历权限。
