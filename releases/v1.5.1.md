# v1.5.1 - 修复搜索详情 sheet 嵌套 bug

## 修复

- 🐛 **搜索详情页 sheet 嵌套 bug**：点击搜索结果时详情页没弹，关闭搜索后又突然弹出。根因是 macOS SwiftUI 同一时刻只支持一个 sheet，详情 sheet 的弹窗请求被压栈。改为把详情内嵌到搜索 sheet 内做条件切换（返回/关闭按钮都是 `selectedSearchResult = nil`），不再弹第二个 sheet。

## 包含的 v1.5.0 变更

### 新增

- 🔍 **日程搜索**：标题栏右侧新增放大镜按钮，弹 sheet 搜索未来 90 天内所有日历中标题包含关键词的日程，按回车触发，点击结果查看详情（日期、时间、所属日历、备注）
- 📅 **单次日程模式**：标题输入区顶部新增 Segmented Picker（"复习计划" / "单次日程"），切换后只在那一天建 1 个日程（不按 [3,7,30] 多次复习），模式切换时复习预览 + 复习间隔区自动隐藏

### 改进

- 🎨 **模式切换过渡动画**：ReviewPreviewSection / IntervalSettingsSection 切换时渐显渐隐（`.scale(0.95) + .opacity`），更自然
- 🎨 **滑动指示器 Picker**：自定义 segmented control，选中段在两段之间用 spring 动画滑动，不再用系统的 segmented style
- 🖱️ **"再建一个"自动滚到输入框**：点"再建一个"后 ScrollView 用 ScrollViewReader 滚到标题输入框
- 🎨 **Picker tint 主题色**：claude 主题下 segmented Picker 与其他 `.borderedProminent` 按钮颜色一致

## 下载

下载 `YijianRili-v1.5.1-macOS.app.zip`，解压后把 `一键日历.app` 拖到「应用程序」文件夹即可。
也可以下载 `.dmg` 镜像获得更顺手的安装体验。

## 系统要求

macOS 14.0+，首次使用会请求日历权限。
