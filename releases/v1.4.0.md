# v1.4.0 - 日程区域支持昨天/今天/明天切换

## 新增

- 📅 **昨天/今天/明天切换**：主界面日程区域右上角新增三段切换器，一键查看任意一天的日程，按钮同时显示短日期（如 `06/22` / `06/23` / `06/24`）
- **动态标题与空态文案**：区域标题和"暂无日程"提示跟随选中日期变化（昨日日程 / 今日日程 / 明日日程）

## 改进

- `CalendarManager.fetchTodayEvents()` 泛化为 `fetchEvents(on:)`，按参数日期查询
- `ReviewViewModel` 新增 `DayType` 枚举 + `selectedDayType` + `displayedEvents` 状态
- 切换状态不持久化，每次启动默认查看今天

## 下载

下载 `一键日历.app.zip`，解压后把 `一键日历.app` 拖到「应用程序」文件夹即可。

## 系统要求

macOS 14.0+，首次使用会请求日历权限。
